The details returned as part of the [GET/api/products](/api-reference/endpoint/products) method in the `product_vendor` field.

| Field             | Description                                                                         |
| ----------------- | ----------------------------------------------------------------------------------- |
| `name`            | The name of the vendor                                                              |
| `bio`             | The full bio of the vendor                                                          |
| `logo`            | Full URL of the vendors logo                                                        |
| `shipping_info`   | Details of the vendors shipping info. This might be blank depending on the vendor   |
| `returns_info`    | Details of the vendors return policy. This might be blank depending on the vendor   |
